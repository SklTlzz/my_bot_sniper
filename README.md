Асинхронный Telegram-бот
Агрегатор рыночных данных с 9 криптобирж через REST API и WebSocket.

Главная идея бота:
Следить за аномалиями на споте токена, c которым начались манипуляции (депозиты с кошельков на биржи после пампа). При нахождении плотностей (аномально больших заявок) в стакане, резких движений, аномальных объемов - отправляется уведомление в Telegram.

Стек: Python, asyncio, aiohttp, aiosqlite, aiogram, numpy.
